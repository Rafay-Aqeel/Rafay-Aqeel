I design and build software end to end, bridging robust backend architecture and clean frontend interfaces. I care about engineering systems that are well-structured, reliable, and built to scale, and I bring AI automation into that work where it earns its place.

## About Me

- **What I do**
  * Architect and ship production-grade software end to end: full-stack web and mobile apps, SaaS platforms, and cloud-deployed systems.
  * Design clean backend architectures and data pipelines built to handle real-world scale and complexity.
  * Build AI automation and agentic workflows that turn manual, repetitive processes into reliable autonomous systems.

- **How I work**
  * I lead with engineering and system design, then use AI-native, agentic workflows to architect and deliver complex systems faster, without trading away quality or control.

- **I'm open to collaborating on**
  * Complex full-stack applications (React, Next.js, Node.js, NestJS), well-architected backend systems, and custom LLM, MCP, and agentic AI integrations for business automation.

- **Ask me about**
  * Full-stack web and mobile development (React / React Native), Node.js and NestJS architecture, RESTful API design, PostgreSQL and data pipelines, cloud deployment, and turning manual workflows into automated, AI-driven systems.
